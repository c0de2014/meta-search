#

# ## Beispiel 1:
# 
# Eintrag:    filtermich.tld
# 
# Erfasst:    filtermich.tld

#             bilder.filtermich.tld
#             animationen.bilder.filtermich.tld
#             banner.bilder.filtermich.tld
#
# Aber nicht: filtermich.domain.tld
#             www.domain.tld/filtermich.tld/Datei.html
#
#
#
# ## Beispiel 2:
# 
# Eintrag:    bilder.filtermich.tld
# 
# Erfasst:    bilder.filtermich.tld
#             animationen.bilder.filtermich.tld
#             banner.bilder.filtermich.tld
#             nervig.banner.bilder.filtermich.tld
#
# Aber nicht: filtermich.tld
#             texte.filtermich.tld
#             bilder.interessant.filtermich.tld
#
#
#
# Achtung: Zur Filterung exakter Hostnamen ist diese Liste *nicht* gedacht!
