# Die Einträge dieser Liste werden nur berücksichtigt, wenn sie als
# Ordner- oder Dateiname in der Pfad-Komponente des überprüften URLs vorkommen.
#
# Als Pfad gilt in diesem Zusammenhang der URL-Teil, der den Dateipfad
# in der vom Webserver nach außen hin präsentierten Verzeichnisstruktur angibt.
# 
# Der Vergleich erfolgt zum Ende hin offen.
# Ein Listeneintrag gilt dann als zutreffend, wenn er bis zu seinem Ende mit einem
# Abschnitt des verarbeiteten URLs übereinstimmt. Ob und wie der URL danach
# noch weitergeht, hat keinen Einfluss auf das Ergebnis.
#
# Die einzelnen Listeneinträge werden nach jedem Schrägstrich mit dem
# überprüften URL verglichen.
#
#
#
#
# ## Beispiel 1:
# 
# Eintrag:    filtermich
# 
# Erfasst:    www.domain.tld/filtermich/
#             www.domain.tld/filtermich/Datei.jpg

#             www.domain.tld/Ordner/filtermich.php?var1=wert1&var2=wert2
#             www.domain.tld/filtermichebenfalls/Datei.jpg
#
# Aber nicht: www.filtermich.tld
#             filtermich.domain.tld
#             www.domain.tld/Ordner/Datei.php?var1=filtermich&var2=wert2
#
#
#
#
